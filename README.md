# JoyCraftMenu

Мод меню JoyCraft для Minecraft Fabric 1.16.5.

Автор: Demon  
Сервер: 45.131.186.59:25565

Для работы нужен Fabric API 0.42.0+1.16    -   https://modrinth.com/mod/fabric-api/versions?g=1.16.5
и Версия fabric-loader-0.14.25-1.16.5   -    https://fabricmc.net/use/installer

ИНСТРУКЦИЯ 

fabric-api-0.42.0+1.16.jar и GriefClientButtons-1.16.5-1.0.0.jar 
надо поместить в файл C:\Users\"user"\AppData\Roaming\.minecraft\mods
И выбрать при запуске игры версию: fabric-loader-0.14.25-1.16.5
